IP是英文Instruction pointer的缩写，意思是“网络之间互连的协议”，IP协议也可以叫做“因特网协议”。

通俗的讲：IP地址也可以称为互联网地址或Internet地址，是用来唯一标识互联网上计算机的逻辑地址。每台连网计算机都依靠IP地址来标识自己，就很类似于我们的电话号码一样，通过电话号码来找到相应的电话，全世界的电话号码都是唯一的。IP地址也是一样。
